Hospital Management System – SQL Queries and Results

Query 1: List all female patients over 20 years old.

SELECT PatientID, Name, Age, Gender, Phone, Address
FROM Patient
WHERE Gender = 'Female' AND Age > 20;

Query 2: List all available (Free) beds in the 'General' ward.

SELECT BedNumber, WardType, Status
FROM bed
WHERE Status = 'Free' AND WardType = 'General';

Query 3: Display all staff members who are Doctors with their specialties and salaries.

SELECT Name, Specialty, Salary
FROM staff
WHERE Role = 'Doctor';

Query 4: List all medicines with a price greater than 100, sorted by price.

SELECT Medicine, Manufacturer, Price
FROM medicine
WHERE Price > 100
ORDER BY Price DESC;

Query 5: Show patient names and their respective diagnoses from the medical records.

SELECT p.Name AS PatientName, m.Diagnosis, m.AdmissionDate
FROM Patient p
INNER JOIN [Medical-record] m ON p.PatientID = m.PatientID;

Query 6: Search for all staff members who have 'Physician' in their specialty.

SELECT Name, Specialty
FROM staff
WHERE Specialty LIKE "*Physician*";
